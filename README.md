# Sunil A M - Portfolio Website

A modern, responsive personal portfolio website for an aspiring Full Stack Developer.

## Features

- **Responsive Design**: Works seamlessly on mobile, tablet, and desktop devices
- **Modern UI/UX**: Clean design with gradient backgrounds and smooth animations
- **Smooth Scrolling**: Navigation with smooth scroll effects
- **Typing Animation**: Dynamic role typing effect in hero section
- **Scroll Reveal**: Sections animate as you scroll down
- **Skill Bars**: Animated progress bars showing skill proficiency
- **Project Showcase**: Display of 4 major projects with hover effects
- **Contact Form**: Functional contact form with validation
- **Social Links**: LinkedIn, GitHub, and Email integration

## Technologies Used

- HTML5
- CSS3 (Flexbox, Grid, Animations)
- JavaScript (Vanilla)
- Font Awesome Icons

## Sections

1. **Navigation Bar** - Sticky navbar with smooth scrolling and mobile hamburger menu
2. **Hero Section** - Introduction with typing animation and call-to-action buttons
3. **About Section** - Personal introduction and technologies being learned
4. **Skills Section** - Categorized skills (Frontend, Backend, Database) with progress bars
5. **Projects Section** - Showcase of 4 projects with descriptions and links
6. **Contact Section** - Contact form with validation and social media links
7. **Footer** - Copyright info, social links, and back-to-top button

## How to Use

1. Open `index.html` in your web browser
2. Customize the content:
   - Update personal information in `index.html`
   - Modify colors in `style.css` (CSS variables in `:root`)
   - Add your project images to the `images/` folder
   - Update social media links

## Customization

### Colors
Edit the CSS variables in `style.css`:
```css
:root {
    --primary-color: #6366f1;
    --secondary-color: #8b5cf6;
    --accent-color: #ec4899;
}
```

### Projects
Update project information in the Projects section of `index.html`

### Social Links
Replace placeholder links with your actual profiles:
- LinkedIn
- GitHub
- Email

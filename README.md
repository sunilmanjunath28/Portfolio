# Sunil A M - Portfolio Website

A professional, clean, and fully responsive portfolio website showcasing my work as a Full Stack Developer.

## Overview

This portfolio has been redesigned with a focus on professional, human-crafted aesthetics. The design emphasizes clarity, readability, and a clean user experience without relying on AI-generated patterns like excessive gradients or generic styling.

## Key Features

- **Professional Design**: Clean, modern layout following industry best practices
- **100% Responsive**: Optimized for all devices (mobile, tablet, desktop)
- **Performance Focused**: Minimal dependencies, fast loading times
- **Accessible**: Semantic HTML5, proper ARIA labels, good contrast ratios
- **Smooth Interactions**: Subtle animations and transitions
- **SEO Optimized**: Proper meta tags and semantic structure

## Technologies Used

- **HTML5**: Semantic markup
- **CSS3**: Custom properties, Flexbox, Grid, modern layouts
- **JavaScript (ES6+)**: Vanilla JS, no frameworks
- **Font Awesome**: Icon library
- **Inter Font**: Professional typography

## Design Philosophy

This portfolio follows professional web design principles:

### Visual Design
- **Color Scheme**: Professional blue (#2563eb) with neutral grays
- **Typography**: Inter font family for clean readability
- **Spacing**: Consistent padding and margins using rem units
- **Borders**: Clean 1px borders instead of heavy shadows
- **Hover Effects**: Subtle transforms and color changes

### User Experience
- **Clear Hierarchy**: Proper heading structure and visual weight
- **Readable Text**: 16px base font, 1.6 line height
- **Touch Targets**: Minimum 44px for interactive elements
- **Feedback**: Visual feedback on all interactions
- **Navigation**: Easy to find and use on all devices

### Technical
- **Mobile-First**: Responsive breakpoints at 768px and 480px
- **Performance**: Optimized images, minimal CSS/JS
- **Accessibility**: Semantic HTML, keyboard navigation
- **Browser Support**: Modern browsers (Chrome, Firefox, Safari, Edge)

## Customization Guide

### Colors
Edit CSS variables in `style.css`:
```css
:root {
    --primary: #2563eb;
    --primary-dark: #1e40af;
    --text-primary: #1f2937;
    --text-secondary: #6b7280;
}
```

### Content
- Update personal information in `index.html`
- Replace profile image in `images/` folder
- Add your project screenshots
- Update social media links

### Styling
- Modify spacing in CSS custom properties
- Adjust breakpoints for responsive design
- Customize animations and transitions
